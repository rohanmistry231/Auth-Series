# TypeScript — Auth Patterns

## Requirements

- Node.js 18+

## Run the Server

```bash
npx tsx server.ts
```

## Run the Client

```bash
npx tsx client.ts
```

## Files

| File | Purpose |
|------|---------|
| `server.ts` | BFF + Token Rotation + Gateway patterns |
| `client.ts` | Demonstrates all three patterns |
