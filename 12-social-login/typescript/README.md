# TypeScript — Social Login

## Requirements

- Node.js 18+

## Run the Server

```bash
npx tsx server.ts
```

Open `http://127.0.0.1:8000` and click "Sign in with Google" or "Sign in with GitHub".

## Run the Client

```bash
npx tsx client.ts
```

## Files

| File | Purpose |
|------|---------|
| `server.ts` | Node.js app + mock social providers |
| `client.ts` | Automated OAuth flow demo |
