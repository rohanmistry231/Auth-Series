# Go — API Key Auth

## Requirements

- Go 1.21+
- No external dependencies

## Run the Server

```bash
go run server.go
```

## Run the Client

```bash
go run client.go
```

## Files

| File | Purpose |
|------|---------|
| `server.go` | Go net/http server with API key management |
| `client.go` | Full lifecycle demonstration client |
