# TypeScript — API Key Auth

## Requirements

- Node.js 18+
- No external dependencies

## Run the Server

```bash
npx tsx server.ts
```

## Run the Client

```bash
npx tsx client.ts
```

## Files

| File | Purpose |
|------|---------|
| `server.ts` | Node.js HTTP server with API key management |
| `client.ts` | Full lifecycle demonstration client |
