# TypeScript — Security Best Practices

## Requirements

- Node.js 18+

## Run the Server

```bash
npx tsx server.ts
```

## Run the Client

```bash
npx tsx client.ts
```

## Files

| File | Purpose |
|------|---------|
| `server.ts` | Rate limiter, security headers, audit log |
| `client.ts` | Demonstrates all protections |
