# Code Examples

Runnable authentication examples organized by language.

## Structure

```
code-examples/
├── node/          # Node.js / TypeScript
│   ├── basic-auth/
│   ├── session-cookies/
│   ├── jwt/
│   ├── oauth2/
│   └── saml/
├── python/        # Python (coming soon)
└── go/            # Go (coming soon)
```

## Quick Start

```bash
cd code-examples/node
npm install
```
